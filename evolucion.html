Versiones clave de HTML

  • HTML 1.0 (1991): Versión experimental, solo enlaces y texto básico.
  • HTML 2.0 (1995): Primer estándar oficial (RFC 1866). Formularios básicos.
  • HTML 3.2 (1997): Soporte para tablas, applets y estilos en línea.
  • HTML 4.01 (1999): Separación de contenido y presentación (CSS), mejor accesibilidad.
  • XHTML (2000): HTML escrito como XML (estricto, bien formado).
  • HTML5 (2014): Semántica rica, multimedia nativa, APIs modernas, offline support.
